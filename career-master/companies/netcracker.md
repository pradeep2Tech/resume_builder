# Netcracker Technology

**Role:** System Architect  
**Period:** July 2025 — Present  
**Location:** Hyderabad, Telangana, India  
**Client / account:** Netcracker enterprise customers

## Overview

Leading modernization of a large-scale Netcracker Enterprise Resource Planning (ERP) platform spanning 100+ modules across HR, Finance, Travel, Procurement, Legal, Inventory, Workforce Management, Revenue Planning, and Workflow domains. Driving cloud-native transformation, Java modernization, enterprise observability, and identity/security architecture across the platform.

## Responsibilities

- Define enterprise modernization roadmap and target-state architecture for 100+ module ERP platform
- Establish architecture governance, deployment standards, and cloud-native reference patterns
- Quantify infrastructure capacity requirements per modernization wave; automate sizing estimates with scripts for repeatable planning
- Perform architecture assessments, domain decomposition, dependency analysis, and migration impact planning
- Drive OpenShift-to-Kubernetes migration and Java 8 → Java 17 upgrade programs
- Define observability, identity, and security architecture standards (OpenTelemetry, OpenSearch, Grafana, ADFS, Keycloak, RBAC, JWT, OPA, MinIO, Vault)
- Partner with product owners, architects, and engineering leadership across Finance, HR, Travel, Workforce, Procurement, and Workflow domains

## Key achievements

- Improved engineering delivery capacity by **40%** through service independence, parallel development, and reduced regression effort
- Successfully delivered **6 critical ERP components** — Customer Delivery Management, Resource Management, HRMS, Travel, Finance, and Workforce — with roadmap for further modernization
- Quantified infrastructure capacity requirements in modernization roadmap; automated sizing estimates with scripts for repeatable deployment planning
- Integrated enterprise observability with OpenTelemetry, OpenSearch, and Grafana; migrated Elasticsearch to OpenSearch with metrics across applications
- Integrated ADFS; established identity and security architecture with Keycloak, RBAC, JWT, and OPA policy standards
- Integrated MinIO (S3-compatible object storage) and HashiCorp Vault for storage and secrets management
