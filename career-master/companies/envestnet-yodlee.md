# Envestnet | Yodlee (Yodlee Infotech Pvt Ltd)

**Role:** Lead Developer / Tech Lead  
**Period:** December 2011 — November 2015  
**Location:** Bangalore, India

## Overview

Led engineering teams of up to 35 across Personal Finance Management (PFM), Data Aggregation Platform (DAP), and internal advisory tools for global banking clients (ANZ, RBC, 22Seven, Nedbank, Nimble, Xero, etc.). Built scalable financial data aggregation, reusable API frameworks, custom prediction algorithms, and text mining solutions.

## Responsibilities

- Lead cross-functional teams on PFM, DAP, DAP Restructure, my Monitor, and Issue Analyzer
- Engineer scalable data aggregation using Java, Selenium, Spring, and Oracle
- Design reusable APIs with advanced design patterns (9+ patterns on PFM alone)
- Develop custom prediction algorithms, static rule engines, and graph algorithms
- Implement text mining (Weka, CSSBox) for unstructured financial data
- Migrate C++ native APIs to Selenium-based crawling agents

## Key achievements

- Built **Issue Analyzer** — reduced manpower by **7 resources** via predictive analytics
- Led **PFM** for tier-1 clients (ANZ, RBC, 22Seven) — 35-member team
- **DAP** — C++ → Selenium migration with ML crawling agents
- **DAP Restructure** — full architectural rewrite with decoupled modules and custom rule engine
- **my Monitor** — customer self-service for service requests (2-person team)
- Spot Award (22Seven); ANZ appreciations
- 2× Innovative Idea Competition selection; 2nd prize Tech Fest; 3rd prize Yodlee Hackathon
