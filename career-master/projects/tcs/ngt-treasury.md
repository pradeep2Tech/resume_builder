# NGT — Next Generation Tool

**Company:** [TCS](../companies/tcs.md)  
**Period:** August 2008 — May 2010  
**Role on project:** Programmer Analyst  
**Team size:** 27  
**Client:** Dhaka Bank (product)

## Problem

Universal banking system needed modernization from PowerBuilder to Java for foreign exchange, trading, money market, and multi-currency/multi-branch operations.

## Solution

- Web-based product integrating Universal Banking System with modern architecture
- PowerBuilder 10.5 → Java migration program
- Rich web UI: Spring, JSP, Servlets, DHTML, CSS, AJAX, Struts
- Presentation layer validations with JavaScript
- PowerBuilder 10.5 code analysis for migration planning
- Shell scripting migration tools converting PowerBuilder logic to JavaScript

## Impact

- Contributed to large-scale (27-member) treasury/banking product migration
- Automated migration scripts accelerating PowerBuilder → Java transition
- Multi-currency, multi-branch FX and money market operations support

## Tech stack

`Java` `Struts` `Spring` `WebLogic 10.3` `PL/SQL` `Unix Solaris` `PowerBuilder 10.5` `Shell scripting` `JavaScript` `AJAX`
