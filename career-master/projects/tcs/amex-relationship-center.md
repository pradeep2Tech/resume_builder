# American Express — Relationship Center (RC)

**Company:** [TCS](../companies/tcs.md)  
**Period:** June 2010 — December 2011  
**Role on project:** Module Lead  
**Team size:** 8  
**Client:** American Express

## Problem

Multiple disconnected American Express web applications needed unification into a single Relationship Center platform with modern frontend, improved performance, and cross-browser compatibility.

## Solution

**Technology migrations:**
- DOJO → jQuery framework
- JSP Scriptlets → JSTL tags
- JSON framework + AJAX (jQuery/JSON standard methods)
- JDK 1.4 → 1.5; WebSphere 5.0 → 6.0
- Maven build tool; YUI Compressor for JS/CSS; PMD checker

**Delivery:**
- Design documentation and work distribution across 8-member team
- Offshore/onsite coordination; cross-vendor synchronization
- Reusable jQuery, JSON, JSTL components
- Maven-based JS/CSS compression automation
- Browser compatibility identification and resolution
- Stored procedure impact analysis; defect management through functional verification

## Impact

- Unified multiple Amex web applications — improved performance, scalability, maintainability
- Task automation and browser compatibility across platforms
- **TCS GEMS Award** recognition for project contribution
