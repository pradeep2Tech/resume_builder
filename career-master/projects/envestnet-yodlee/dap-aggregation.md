# DAP — Data Aggregation Platform

**Company:** [Envestnet Yodlee](../companies/envestnet-yodlee.md)  
**Period:** May 2014 — November 2014  
**Role on project:** Team Lead / Requirement Analyst  
**Team size:** 4  
**Client:** Yodlee — all customers

## Problem

Financial data aggregation relied on legacy C++ native APIs. Yodlee needed a modern Selenium/Chrome-based crawling platform with machine learning agents for intelligent data extraction.

## Solution

- Built aggregation platform on Chrome and Selenium drivers
- Migrated all existing C++ native APIs to Selenium APIs
- Applied machine learning to develop intelligent crawling agents
- Common APIs supporting multiple financial containers
- Common APIs for automated code generation
- Text mining with Weka and CSSBox for unstructured data

## Impact

- Modernized crawling infrastructure from native C++ to Selenium
- Enabled ML-driven intelligent agents for financial data extraction
- Reusable APIs reduced development effort across financial container types
